## 📖 项目介绍

### 这个项目是什么

**咖啡商城小程序**是一个面向咖啡店的**扫码点单系统**，包含以下核心功能：

- 🛒 **扫码点单** - 顾客扫桌面二维码进入小程序，浏览商品、加入购物车、在线支付
- 👨‍💼 **商家后台** - 实时接收订单、管理商品、制作管理、订单发货
- 💬 **社区功能** - 用户分享咖啡体验、发布图文笔记、互动交流
- 🎨 **AI图片润色** - 商家上传商品图片，AI自动优化图片质量
- 💰 **多种支付** - 支持余额支付、收钱吧支付

### 技术栈

**后端**
- Spring Boot 2.x
- MyBatis
- MySQL 8.0
- Redis
- Shiro（权限管理）
- Quartz（定时任务）

**前端**
- uni-app（小程序）
- Vue.js
- Thymeleaf（后台管理页面）

**第三方服务**
- 微信小程序
- 腾讯云COS（对象存储）
- 收钱吧支付（计划接入）
- AI图片处理服务（计划接入）

## 🚀 项目怎么使用

### 环境要求

- JDK 1.8+
- MySQL 8.0+
- Redis 3.0+
- Maven 3.6+
- Node.js 14+
- 微信开发者工具

### 后端启动步骤

#### 1. 克隆项目
```bash
git clone https://github.com/你的用户名/coffee-shop.git
cd coffee-shop
```

#### 2. 创建数据库
```bash
# 登录MySQL
mysql -u root -p

# 创建数据库
CREATE DATABASE coffee_shop DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入数据库脚本
use coffee_shop;
source sql/coffee_shop.sql;
```

#### 3. 修改配置文件

编辑 `src/main/resources/application.yml` 和 `application-druid.yml`

```yaml
# 数据库配置
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/coffee_shop?useUnicode=true&characterEncoding=utf8
    username: root
    password: 你的数据库密码
    
# Redis配置
redis:
  host: localhost
  port: 6379

# 腾讯云COS配置（建议使用环境变量）
cos:
  secret-id: ${COS_SECRET_ID}
  secret-key: ${COS_SECRET_KEY}
```

#### 4. 启动项目
```bash
# 使用Maven启动
mvn clean install
mvn spring-boot:run

# 或在IDE中运行 RuoYiApplication.java
```

#### 5. 访问后台
```
URL: http://localhost:8080
账号: admin
密码: admin123
```

### 小程序端启动步骤

#### 1. 进入小程序目录
```bash
cd RuoYi-AbuCoder-UniAppWx/Ruoyi-AbuCoder-UniApp-WX
```

#### 2. 修改API地址
编辑 `utils/apiconfig.js`
```javascript
export default {
  baseUrl: 'http://localhost:8080'  // 改为你的后端地址
}
```

#### 3. 导入微信开发者工具
- 打开微信开发者工具
- 导入项目目录
- 填写AppID
- 编译运行

## ⚠️ 已知问题与风险

### 🔴 高优先级问题（P0）

#### 1. 支付回调可能丢失
- **发生概率**：约2-3%
- **影响**：用户付款但订单未更新
- **解决方案**：三重补偿机制（回调+轮询+定时任务）

#### 2. 订单金额可能被篡改
- **发生概率**：低但危害大
- **影响**：商家经济损失
- **解决方案**：后端重新计算所有金额

#### 3. 并发可能导致超卖
- **发生概率**：中（促销时高发）
- **影响**：库存-1但卖出2件
- **解决方案**：SQL WHERE条件防并发

#### 4. 密钥明文存储
- **现状**：配置文件直接写密钥
- **改进**：使用环境变量

#### 5. 违规内容可能传播
- **影响**：小程序可能被封
- **解决方案**：微信内容安全API + 人工审核

**详细问题清单**见：[问题梳理与风险清单](docs/问题梳理与风险清单.md)
